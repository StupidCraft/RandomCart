## Compiling
1. #### Clone RandomCart
```sh
git clone https://github.com/StupidCraft/RandomCart.git
```
2. #### Change directory to RandomCart
```sh
cd RandomCart
```
4. #### Build using Gradle
```sh
./gradlew shadowJar
```

You will find a compiled RandomCart Jar file in `build/libs/`.
